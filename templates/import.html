CSV Format Requirements

The CSV file should contain the following columns (semicolon-separated):

  • Numéro de compte (Account Number)
  • Nom du compte (Account Name)
  • Compte contrepartie (Counterparty Account)
  • Numéro de mouvement (Transaction Number)
  • Date comptable (Accounting Date - DD/MM/YYYY)
  • Date valeur (Value Date - DD/MM/YYYY)
  • Montant (Amount with comma as decimal separator)
  • Devise (Currency)
  • Libellés (Description)
  • Détails du mouvement (Details)
  • Message

Duplicate transactions will be automatically skipped.
